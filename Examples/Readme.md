# Examples

This folder contains example parameter configurations for running experiments on the following datasets:

- Toy Regression dataset with and without noise

- NYUv2 [[1]](#1)

- CIFAR10 with and without flipped labels [[2]](#2)

- Multi-MNIST [[3]](#3)
